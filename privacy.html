Smart Scraper 隐私政策

生效日期：2026 年 5 月 13 日

Smart Scraper 的用途是帮助用户在当前访问的网页中识别、提取、整理并导出结构化数据。扩展只会在用户主动点击插件或启用具体功能后处理当前页面数据，不会出售用户数据，也不会将用户数据用于广告投放、画像或与本扩展功能无关的目的。

1. 我们处理哪些数据

Smart Scraper 可能会在用户主动使用相关功能时处理以下数据：

  • 当前网页内容：用于列表检测、页面信息提取、元素选择、数据预览和导出。
  • 当前网页中的 JSON 接口响应：仅在用户启用“请求抓取”功能时捕获，用于识别、展示和导出结构化数据。
  • 用户选择或生成的采集结果：包括字段、表格行、页面标题、链接、图片等由用户主动提取的数据。
  • 本地设置与凭据：包括登录令牌、令牌过期时间、用户信息、GLM API Key 和自动翻页临时状态。
  • 登录相关信息：当用户选择登录或使用需要登录的自动翻页功能时，会处理手机号、验证码或密码等登录请求所需信息。

2. 数据如何使用

上述数据仅用于提供 Smart Scraper 的用户可见功能，包括：

  • 检测网页中的重复列表、表格、卡片和内容区域。
  • 根据用户手动选择的元素生成结构化字段。
  • 捕获当前页面的接口数据并整理为表格。
  • 支持自动翻页、滚动加载和请求模式下的数据追加采集。
  • 导出 CSV 或 JSON 文件，或将 JSON 复制到剪贴板。
  • 保持用户登录状态、保存本地扩展设置，以及在页面跳转后恢复用户正在进行的采集任务。
  • 在用户配置 GLM API Key 并点击 AI 智能分析后，辅助分析页面列表项的字段结构。

3. 数据是否会共享或传输给第三方

Smart Scraper 不会出售用户数据。扩展只会在以下场景中传输数据：

  • 登录、验证码发送和用户资料校验请求会发送到企微瓣服务接口（https://www.qiweiban.com/api）。
  • 当用户主动配置 GLM API Key 并点击 AI 智能分析时，扩展会将页面列表项的 HTML 样本发送到智谱 AI 接口（https://open.bigmodel.cn/），用于分析字段结构。
  • 除上述用户主动触发的功能外，Smart Scraper 不会将当前网页内容、采集结果或接口响应上传到开发者服务器或其他第三方服务。

4. 本地存储与保留期限

Smart Scraper 使用 Chrome 本地存储保存必要数据，包括登录令牌、令牌过期时间、用户信息、GLM API Key 和自动翻页临时状态。采集结果主要保存在当前浏览器会话中，用户可以清除或导出。

用户可以通过退出登录、删除 GLM API Key、清除浏览器扩展数据或卸载扩展来删除本地保存的数据。扩展本身不会在远程服务器保存用户采集的网页数据。

5. 权限说明

  • activeTab：在用户点击插件或启动侧边栏功能后访问当前标签页，用于在当前网页中检测和提取数据。
  • scripting：向当前网页注入扩展自带的内容脚本和样式，用于侧边栏、元素高亮、列表检测、手动选择和请求捕获。
  • storage：在用户浏览器本地保存扩展设置、登录状态、GLM API Key 和自动翻页临时状态。
  • webNavigation：在自动翻页或页面跳转后重新注入内容脚本，并恢复用户正在进行的采集任务。

6. 数据安全

Smart Scraper 的可执行代码随扩展包一起分发，不会加载或执行远程托管代码。扩展与外部服务通信时使用 HTTPS。请用户仅在有权访问和使用相关数据的网站上使用本扩展，并妥善保管自己的 API Key 和账号信息。

7. 用户控制

用户可以随时停止采集、清除当前数据、导出数据、退出登录、删除 API Key、禁用或卸载扩展。用户也可以在 Chrome 浏览器的扩展管理页面中管理或移除 Smart Scraper 的本地数据。

8. 儿童隐私

Smart Scraper 面向一般生产力场景，不面向儿童提供服务，也不会有意收集儿童个人信息。

9. 政策更新

如果扩展功能、数据处理方式或第三方服务发生变化，本隐私政策会相应更新，并在本页面展示新的生效日期。
